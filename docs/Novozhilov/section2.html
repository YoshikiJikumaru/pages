共役網とCombescure変換

$\br^3$ 内の曲面 $\Sigma$ が局所的に写像 $\brr: D \to \br^3$ によってパラメトライズされているものとする． ここで $D \subset \br^2$ は領域であり，座標を $(x, y)$ で表す． $\brr$ が双曲型方程式 \[ \tag{1} \brr_{xy} = a \brr_x + b \brr_y, \] を満たすとき，座標 $(x, y)$ を共役座標 (conjugate coordinates)，アイソパラメトリックライン $x = const.$, $y=const.$ により曲面上に生成されるネットを 共役網 (conjugate net) と呼ぶ．

曲面論における基本的な記号

本章では，曲面論から基本的な関係式について復習する． 読者には微分幾何学とベクトル解析についての基礎知識を仮定する． すべての滑らかな曲面は，デカルト座標において$3$つの方程式によって決定される： \[ \tag{2.1} x = f_1 (\a_1, \a_2), \quad y = f_2 (\a_1, \a_2), \quad z = f_3 (\a_1, \a_2). \] ここで$f_1, f_2, f_3$は$2$変数$\a_1, \a_2$に関して連続な一価関数である． これらのパラメータの範囲は，曲面のすべての点が値のペア $(\a_1, \a_2)$ と一対一対応があるような範囲に制限して考える． このとき，これらのパラメータは考えている曲面の曲線座標 (curvilinear coordinates)と呼ばれる． 式\eqref{eq:2.1}におけるパラメータ $\a_2$ を連続する異なる値に固定し，$\a_1$ を動かすとき，曲面上に曲線の族が得られ，これらを座標線 $\a_1$ と名付ける． 同様に，$\a_1$ を固定するときに $\a_2$ を動かすと，第2の曲線の族が得られ，これを座標線 $\a_2$ と呼ぶ． 上記の変数の範囲の制約条件によって，曲面上の任意の点に対し，その点を通る曲線がこれら$2$つの曲線族の中からちょうど$2$つだけ定まる． よって，曲面上のすべての点は$2$つの座標線$\a_1, \a_2$の共通部分と考えることができる． 一つの曲面に対し，関数$f_1, f_2, f_3$のとり方は無限に存在し，とり方に応じて曲面上の座標線の形は変わる． ところが，これらの候補は等しく有用であるというわけではない． 一般に，これらの選択肢のうち，最も功利的であるのは曲線座標系の中で曲面に対する公式が最も簡単な形となるものである． そのような座標系は，座標曲線の$2$つの族が，同時に曲面の主曲率線でもあるようにとることである． そのような座標系は直交座標である，というのも曲面上の主曲率の方向は互いに直交するからである． 以下では，この曲線座標が導入されているものとする． $3$つのスカラー方程式\eqref{eq:2.1}は，単一のベクトル方程式 \[ \tag{eq:2.2} \br = \br (\a_1, \a_2) \] に対応し，座標軸$x, y, z$への射影が\eqref{eq:2.1}で与えられる． 変数$\a_1, \a_2$に関する$\br$の偏微分を計算しよう： \[ \tag{eq:2.3} \br_{\a_1} = \frac{\d \br}{\d \a_1}, \quad \br_{\a_2} = \frac{\d \br}{\d \a_2}. \] ここで，ベクトル$\br_{\a_1}, \br_{\a_2}$は曲面の各点において座標線$\a_1, \a_2$にそれぞれ接する． 結果として，曲線座標の一つパラメータの増加に従って，対応する座標線に対応する方向における曲面上のベクトル$\br$が動く． よって，ベクトル \[ \tag{eq:2.4} d \s = \frac{\d \br}{\d \a_1} d \a_1 + \frac{\d \br}{\d \a_2} d \a_2 \] は，曲面上の点$(\a_1, \a_2)$, $(\a_1 + d\a_1, \a_2 + d\a_2)$を結ぶ線分の大きさと向きに対応する． この線分の長さの二乗は， \[ \tag{eq:2.5} |d \s|^2 = |\br_{\a_1}|^2 d\a_1^2 + 2 \< \br_{\a_1}, \br_{\a_2} \> d\a_1 d\a_2 + |\br_{\a_2}|^2 d \a_2^2 \] で与えられる． 記号$|\cdot|^2$はベクトルとそれ自身との内積を表し，ベクトルの長さの二乗に等しい． 同様に，記号$\< \cdot, \cdot \>$は$2$つのベクトルの内積を表す． 新しく次の記号を導入する： \[ \begin{aligned} |\br_{\a_1}|^2 &= A_1^2 = \left( \frac{\d x}{\d \a_1} \right)^2 + \left( \frac{\d y}{\d \a_1} \right)^2 + \left( \frac{\d z}{\d \a_1} \right)^2, \cr |\br_{\a_2}|^2 &= A_2^2 = \left( \frac{\d x}{\d \a_2} \right)^2 + \left( \frac{\d y}{\d \a_2} \right)^2 + \left( \frac{\d z}{\d \a_2} \right)^2. \tag{eq:2.6} \end{aligned} \] 曲線座標の直交性の仮定により， \[ \tag{eq:2.7} \< \br_{\a_1}, \br_{\a_2} \> = 0 \] が成り立つ． よってこのとき式\eqref{eq:2.5}は次のように表せる： \[ \tag{eq:2.8} ds^2 = A_1^2 d\a_1^2 + A_2^2 d\a_2^2. \] よって曲線座標のうち一つのパラメータが動くとき， \[ \tag{eq:2.9} ds_1 = A_1 d\a_1, \quad ds_2 = A_2 d\a_2 \] なることを得る． ここで$ds_1, ds_2$はそれぞれ，曲線座標の増分$d \a_1, d\a_2$に対応する座標線$\a_1, \a_2$に沿う弧長の増分である． $A_1, A_2$はそれぞれ{\bf Lam\'{e}パラメータ}と呼ばれる． 以降では，曲面上の点の関数であるすべてのベクトルは，曲面に接する座標線$\a_1, \a_2$の方向と，法線方向に射影して考える． これらのreference axesは，曲面上の一点から別の点へ移るときに変化するので，これらのベクトルの$\a_1, \a_2$に関する微分を観察する必要がある． 考えている系に対し，すべてのベクトル$\T$が \[ \tag{eq:2.10} T = T_{\a_1} \be_1 + T_{\a_2} \be_2 + T_n \be_n \] と表せる． ここで $T_{\a_1}, T_{\a_2}, T_n$ はスカラーであり，$\be_1, \be_2, \be_n$ は直交する単位ベクトル \[ \tag{eq:2.11} \be_1 = \frac{1}{A_1} \br_{\a_1}, \quad \be_2 = \frac{1}{A_2} \br_{\a_2}, \quad \be_n = \be_1 \times \be_2, \] である． ゆえに，ベクトルに対する微分律の研究は，この単位ベクトル\beqref{eq:2.11}に対するそれに帰着させることができる． \beqref{eq:2.11}の最後の式において，シンボル$\times$はベクトル積を表す． 以降，系$\be_1, \be_2, \be_n$は右手系であると仮定する． さらに，$\be_n$は曲面の凸方向を向く（あるいは曲率半径の符号が等しくないときは，負の曲率半径の中心を向く）ようにしておく． これらの条件は互いに矛盾することはない． というのも，$\a_1,
