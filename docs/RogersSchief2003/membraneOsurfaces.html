本ページでは「膜O曲面 (membrane O surfaces)」についてまとめる．

共役網とCombescure変換

$\R^3$ 内の曲面 $\Sigma$ が局所的に写像 $\bR: D \to \Sigma \subset \R^3$ によってパラメトライズされているものとする． ここで $D \subset \R^2$ は領域であり，座標を $(x, y)$ で表す． $\bR$ が双曲型方程式 \[ \tag{1.1} \bR_{xy} = a \bR_x + b \bR_y, \] を満たすとき，座標 $(x, y)$ を共役座標 (conjugate coordinates)，アイソパラメトリックライン $x = const.$, $y=const.$ により曲面上に生成されるネットを 共役網 (conjugate net) と呼ぶ． いま以下のようにしてポテンシャル $H$, $K$ を導入する: \[ \tag{1.2} a = (\ln A_1)_y = \frac{(A_1)_y}{A_1}, \quad b = (\ln A_2)_x = \frac{(A_2)_x}{A_2}. \] さらにこれらのポテンシャル $A_1, A_2$ を用いて，接ベクトル $\bR_x$, $\bR_y$ を次のように分解する: \[ \tag{1.3} \bR_x = \bX A_1, \quad \bR_y = \bY A_2. \] この段階では $\bX$, $\bY$ が単位接ベクトルであることなどは要請しないものとする． このとき，2階の双曲型方程式 (1) は以下の1階線型方程式系へ還元される: \[ \tag{1.4} \bX_y = p \bY, \quad \bY_x = q \bX. \] ここで $p$, $q$ は次式を満たすものとして定義される: \[ \tag{1.5} (A_1)_y = p A_2, \quad (A_2)_x = q A_1. \] 式 (1.5) は線型方程式系 (1.4) の随伴 (adjoint) とみなすことができる． 逆に，与えられたベクトルとスカラーの組 $(\bX, \bY, A_1, A_2)$ および函数 $p, q$ が，関係式 (1.4), (1.5) によって関係づけられているとき，線型方程式系 (3) は両立条件 $\bR_{xy} = \bR_{yx}$ を満たし，曲面 $\Sigma \subset \R^3$ が得られる． いま (1.5) の別の解 $(H_*, K_*)$ が得られたとするとき，線型方程式系 \[ \tag{1.6} (\bR_*)_x = H_* \bX, \quad (\bR_*)_y = K_* \bY, \] は両立条件 $(\bR_*)_{xy} = (\bR_*)_{yx}$ を満たし，別の曲面 $\Sigma_* \subset \R^3$ が定まる． 曲面 $\Sigma_*$ を $\Sigma$ の Combescure変換 (Combescure transformation) と呼ぶ． Combescure変換により，対応する点では (座標線に沿った) 接ベクトルが平行である． さらにCombescure変換は共役網の条件 (1.1) を保存することが確かめられる．

E. Combescure, Sur les déterminants fonctionnels et les coordonnées curvilignes, Annales scientifiques de l'École Normale Supérieure, Serie 1 (1867) 4, 93-131.

直交共役座標 (曲率線座標)

以下では $\bX$, $\bY$ はいずれも単位ベクトルとし，互いに直交するものと仮定する: \[ \tag{7} \bX \cdot \bX = 1, \quad \bY \cdot \bY = 1, \quad \bX \cdot \bY = 0. \] このとき座標 $(x, y)$ を直交共役座標または曲率線座標と呼ぶ． 曲面の単位法ベクトルを $\bN = \bX \times \bY$ で表す． このとき (4) より \[ \tag{8} \bN_x \cdot \bY = 0, \quad \bN_y \cdot \bX = 0, \] となるので， \[ \tag{9} \bN_x = \bX H_\circ, \quad \bN_y = \bY K_\circ, \] となる $H_\circ$, $K_\circ$ が存在する． いま，関係式 \[ \tag{10} - \bN_x \cdot \bX = - \frac{H_\circ}{H} = \kappa_1, \quad - \bN_y \cdot \bY = - \frac{K_\circ}{K} = \kappa_2, \] により定義される $\kappa_1$, $\kappa_2$ を主曲率と呼ぶ． 定義より \[ H_\circ = - \kappa_1 A_1, \quad K_\circ = - \kappa_2 A_2 \] であり，このとき式 (9) は \[ \tag{11} \bN_x = - \kappa_1 A_1 \bX = - \kappa_1 \bR_x, \quad \bN_y = - \kappa_2 A_2 \bY = - \kappa_2 \bR_y, \] と表される． この式をWeingartenの公式と呼ぶ． Weingartenの公式はまた，球面 $\Sigma_\circ$ が曲面 $\Sigma$ のCombescure変換であることを示している． このとき，曲面の第一・第二基本形式は以下のように表される: \[ I = A_1^2 \, dx^2 + A_2^2 \, dy^2, \quad II = \kappa_1 A_1^2 \, dx^2 + \kappa_2 A_2^2 \, dy^2. \] Combescure変換は，曲率線座標も保つことに注意．

曲率線座標におけるGauss-Weingartenの公式，Gauss-Mainardi-Codazzi方程式

正規直交枠 $\Phi = (\bX, \bY, \bN)$ を用いて，Gauss-Weingartenの公式は線型方程式系 \[ \Phi_x = \Phi \begin{pmatrix} 0 & p & H_\circ \\ - p & 0 & 0 \\ - H_\circ & 0 & 0 \end{pmatrix}, \quad \Phi_y = \Phi \begin{pmatrix} 0 & q & 0 \\ - q & 0 & K_\circ \\ 0 & - K_\circ & 0 \end{pmatrix}. \] で表される． Gauss-Weingartenの公式に対する両立条件 $(\Phi_x)_y = (\Phi_y)_x$ はGauss方程式 \[ p_y + q_x + H_\circ K_\circ = 0, \] およびMainardi-Codazzi方程式 \[ (H_\circ)_y = p K_\circ, \quad (K_\circ)_x = q H_\circ, \] である． 特にGauss曲率を$\mathcal{K} = \kappa_1 \kappa_2$で定めると，Gauss方程式から \[ \mathcal{K} = - \frac{1}{A_1 A_2} \left( \left( \frac{(A_1)_y}{A_2} \right)_y + \left( \frac{(A_2)_x}{A_1} \right)_x \right), \] が成り立つ． すなわちGauss曲率は第一基本形式の係数のみから定まる (Gaussの驚異の定理)．

シェルの膜理論における釣り合い式: 等張力曲面と線型Weingarten曲面

シェル理論およびシェルの膜理論についての詳細はこちらのページを参照． シェル中央面を $\bR$ で表す． 膜の形状および所定の外力が与えられたとき，膜の釣り合い式は面内応力に対するwell-determinedな線型系をなす． ところが，（物理的な）条件が面内応力に課されるとき，釣り合い式はGauss-Mainardi-Codazzi方程式に制約を課し，すなわち膜の幾何学的形状にも制約を与える．
いま，法線方向に作用する荷重 $q_n$ に対し，膜応力を $T_1$, $T_2$，面内せん断を $S$ と表すと，膜の釣り合い式は以下のように表される: \[ \begin{aligned} (A_2T_1)_x + (A_1S)_y + (A_1)_y S - (A_2)_x T_2 &= 0, \cr (A_2S)_x + (A_1T_2)_y + (A_2)_x S - (A_1)_y T_1 &= 0, \cr \kappa_1 T_1 + \kappa_2 T_2 + q_n &= 0 \end{aligned} \] ここで「主曲率線が主応力線に一致する」という要請を課す（応用上重要である）． これは $S = 0$ とおくことに対応し，このとき釣り合い式は \[ (A_2T_1)_x = (A_2)_x T_2, \quad (A_1T_2)_y = (A_1)_y T_1, \quad \kappa_1 T_1 + \kappa_2 T_2 + q_n = 0, \] と表される．
いま，最も簡単な場合として「等張力」で釣り合う場合，すなわち \[ T_1 = T_2 = c = {\rm const.}, \] の場合を考える． このとき面内釣り合い式は自明に成り立ち，面外釣り合い式は \[ c (\kappa_1 + \kappa_2) + q_n = 0 \iff \cH = \frac{\kappa_1 + \kappa_2}{2} = - \frac{q_n}{2c}, \] となる． ここで$\cH = (\kappa_1 + \kappa_2)/2$は曲面の平均曲率である．
例えば$q_n$が内圧と外圧の差（Laplace圧）で一定であったとするとき， 釣り合い形状は平均曲率一定 (Constant mean curvature, CMC) 曲面となる． この式はYoung-Laplace方程式と呼ばれる． ちなみに，それ以前にNavier (1826) によってその原型はすでに発見されている．
さて，このように面内応力に条件を課すことで膜の幾何学的形状に制約が課される様子をもう少しみてみよう．

シェルの膜理論における釣り合い式: 膜O曲面
